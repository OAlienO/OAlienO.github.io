## MetaMask

好用的管理錢包的插件，幫你簽署交易  
在使用其他線上工具或網站的時候，那些網站就只是包好交易內容然後呼叫 MetaMask 用你的私鑰簽署交易

## [MEW](https://www.myetherwallet.com/)

可以發錢或是創建合約跟合約互動，和以太鍊溝通的前端介面  
不過這個工具有些 bug，比如合約呼叫 payable 函式的時候，錢發不出去
