# What's new in python 3.7

## Data Classes

```python
from dataclasses import dataclass

@dataclass
class A:
    x: int
    y: int

a = A(1, 2)
b = A(1, 2)

print(a)
print(a == b)
```

```tab="output"
A(x=1, y=2)
True
```

根據 [annotations](/programming/python/tricks/#annotations) 自動幫你產生 `__init__`, `__repr__`, `__eq__`
